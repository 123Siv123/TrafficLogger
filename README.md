# TrafficLogger

TrafficLogger is a Flask-based web application for logging, managing, and tracking traffic-related information. It provides an easy-to-use interface for recording traffic data and viewing stored records.

## Features

- Log traffic records
- View and manage stored data
- User-friendly web interface
- Flask backend with database support

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite (or your database)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/123Siv123/TrafficLogger.git
   ```

2. Navigate to the project directory:
   ```bash
   cd TrafficLogger
   ```

3. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   ```

   Windows:
   ```bash
   venv\Scripts\activate
   ```

4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

5. Run the application:
   ```bash
   flask run
   ```

6. Open your browser and visit:
   ```
   http://127.0.0.1:5000/
   ```
